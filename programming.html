Programming Projects

The Boston Art Dealers Association Website

For a group project in Web Programming class, we redesigned the website of the Boston Art Dealers Association (BADA).

I recognized that the current website of BADA is not very modern, and considering that the main audience of the website would be an art enthusiast, I realized that redesigning their website would be an interesting project.

As a lead designer and a front-end developer, I designed the look and feel of the entire website and the information architecture of the website.

I also successfully implemented a 3D carousel for the gallery page using CSS 3D, JavaScript, and jQuery animation that made the website much sleeker and more professional.

Immersive Art Experience

As an individual project, I am creating a program where users can explore art exhibitions simply through eyegaze.

Most art gallery websites only allow users to click their art to access the information, so they fail to offer immersive experiences.

I think allowing users to view different artworks only by gazing at different directions can not only allow new interactions with online galleries but also give new opportunities for users who cannot use their limbs.

I am implementing this program with Webgazer.js, jQuery, HTML, CSS, and JavaScript.

Aiden's Music Chart

The Grep

As a final project for Data Structures class, I designed and built a program that efficiently indexes all words in files within a directory.

I used a dynamic hash table of linked lists in C++ to handle search queries much faster than the reference implementation made by the course TA.
